##Файловое облако

#### Приложение позволяет:
* Загружать в облако файлы, размером не более 500 МБ;
* Скачивать файлы из облака;
* Удалять файлы из облака.

Для работы приложения необходимо запустить от имени администратора сервер - host.exe, и затем, также от имени администратора, client.exe.

Для хранения файлов используется база данных. Можно использовать PostgeSQL, либо в режиме InMemory. Для настройки необходимо изменить подключение в Cloud\MyCloud\Model\CloudFileContext.cs: testDb - режим InMemory, myConn - PostgreSQL. Для PostgreSQL также дополнительно необходимо изменить параметры подключения к БД. В конфигурационном файле нужно изменить connectionString. 

Клиент работает на WinForms, сервер - на WCF, для доступа к БД используется Entity Framework.
